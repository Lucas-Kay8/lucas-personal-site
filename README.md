## 🛠️ 技术栈 (Tech Stack)

*   **Core**: HTML5, CSS3, JavaScript (ES6+)
*   **Style**: Bright Apple Style (Glassmorphism, Layered Shadows)
*   **No Frameworks**: 坚持零依赖，追求极致的轻量化与加载速度。

## 🚀 部署指南

本项目适配 **GitHub Pages**。
1.  进入 GitHub 仓库的 **Settings**。
2.  在左侧菜单找到 **Pages**。
3.  在 **Build and deployment** 下的 **Branch** 选项中，选择 `main` 分支。
4.  点击 **Save**。
5.  等待约 1 分钟，刷新页面即可看到生成的 URL。
